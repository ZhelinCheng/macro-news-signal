## 信号级别定义

| 级别 | 说明 | 持续时间 |
|------|------|---------|
| **Trend-Setting** | 趋势级，影响未来数周至数月 | 长期 |
| **Sustained** | 持续级，影响未来数天至数周 | 中期 |
| **Flash** | 瞬时冲击，仅当时段有效 | 短期 |

## 资产信号定义

| 信号 | 含义 |
|------|------|
| 📈 买入 | 预计上涨，适合做多 |
| 📉 卖出 | 预计下跌，适合做空 |
| ➡️ 持有 | 预期震荡，适合观望 |

## 冲击强度定义

| 强度 | 说明 |
|------|------|
| 极高 | 重大黑天鹅，对市场有决定性影响 |
| 高 | 重要事件，能引发显著市场波动 |
| 中 | 常规事件，可能引发短期波动 |
| 低 | 次要事件，市场影响有限 |
